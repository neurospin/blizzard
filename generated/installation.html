Installing blizzard¶

This tutorial will walk you through the process of intalling BLIZZARD…

Installing a stable version¶

Install the python package with pip¶

Install the package without the root privilege

>>> pip install --user python-blizzard

Install the package with the root privilege

>>> sudo pip install python-blizzard
